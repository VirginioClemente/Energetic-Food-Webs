# Curated Data Availability

The curated workbooks used directly by the analysis scripts are not included in
this public repository. They are available from the corresponding author upon
reasonable request.

To rerun the full pipeline, place the requested files in this folder with the
following names:

- `Dataset_analysis_CV.xlsx`
- `individual_plot_tot_biomass_date0_resp_moist.xlsx`
- `individual_plot_tot_biomass_date0_resp_moist_mean_over_plot.xlsx`

The markdown files in this folder document how these curated workbooks were
constructed from the raw source datasets.
