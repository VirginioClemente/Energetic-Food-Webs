# Data Availability

The raw datasets required by the analysis are not included in this public
repository. They are available from the corresponding author upon reasonable
request.

To rerun the full pipeline, place the requested files in this folder with the
following names:

- `topology.txt`
- `data_set_published.xlsx`
- `merged_PublishedData.xlsx`
- `FoodWeb_vectorsFinal.xlsx`
- `individual_plot_tot_biomass_flux.xlsx`
